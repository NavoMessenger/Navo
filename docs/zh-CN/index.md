# Navo

独立、非官方的 **Android / Windows / macOS** 跨平台即时通讯客户端。使用现有 Telegram 账号登录 — 在你熟悉的 Telegram 网络上，体验更轻量的客户端。

**[下载](download.html)** · **[Google Play](https://play.google.com/store/apps/details?id=im.navo.app)** · [GitHub](https://github.com/NavoMessenger/Navo) · [English]({{ site.baseurl }}/)

> Navo **与 Telegram 无隶属、背书或关联关系**。请自行承担风险，并遵守 Telegram 服务条款。

## 功能

- 聊天列表与实时会话更新
- 贴纸与动态贴纸（`.tgs` / `.webm`）
- 语音消息、图片、文件与媒体
- 投票、清单与位置分享
- 联系人、个人资料与 Stories 风格动态
- 一对一语音与视频通话
- 浅色 / 深色主题与可自定义应用图标
- 关键词屏蔽等注重隐私的功能
