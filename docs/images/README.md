# Image assets

Place marketing and site images here. Paths below are relative to this folder.

## Banner

| File | Use | Suggested size |
|------|-----|----------------|
| `banner.png` | README hero + site favicon fallback | **720×200** (or wider, ~720px wide) |

Referenced from repository root as `docs/images/banner.png`.

## Screenshots

Put product screenshots in `screenshots/`:

| File | Suggested size |
|------|----------------|
| `screenshots/chat.png` | ~360–540px wide (phone UI) |
| `screenshots/list.png` | same |
| `screenshots/settings.png` | same |

After adding files, uncomment the screenshot blocks in:

- `README.md` / `README.zh-CN.md`
- `docs/index.md` / `docs/zh-CN/index.md`

## Optional

- Platform install helpers (SmartScreen / Gatekeeper screenshots) for the download guide
- Social preview **1280×640** for GitHub repository Settings → Social preview

## Notes

- Prefer PNG or WebP; keep file sizes reasonable for GitHub.
- Do not commit proprietary third-party trademarks or assets that are not yours.
